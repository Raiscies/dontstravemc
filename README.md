## 📖 关于本模组

我的世界、泰拉瑞亚、饥荒——国内玩家口中的"沙盒三巨头"。它们的共同点是什么？**玩家可见的所有元素或对象，均可以等自由度地进行交互**。

永恒领域模组致力于将饥荒巨人国的硬核生存内容参考饥荒联机版进行完善，搬运到我的世界中，构建一个真正的永恒大陆。不同于简单的内容照搬，本模组深度考虑了与我的世界原版的契合度，力求在保留饥荒核心体验的同时，创造出一种既不同于饥荒又不同于我的世界的独特游戏体验。

### 🎯 设计理念

- **硬核生存**：还原饥荒的生存困境，包括精神值系统、温度机制、四季变化、怪物入侵等
- **深度融合**：不是简单照搬，而是深度考虑与我的世界原版的互动和兼容性
- **沙盒自由**：保持高自由度的交互体验，让玩家能够创造性地解决生存问题
- **丰富内容**：从生物群系到 BOSS，从科技树到烹饪系统，提供完整的游戏循环

## 🎲 游戏体验

### 饥荒的硬核难度来源

本模组致力于还原饥荒的核心难度设计：

- **三维管理**：血量不会自然恢复，必须通过食物、睡眠或药品恢复
- **温度挑战**：冬季严寒与夏季酷暑，需要提前准备保暖/降温装备
- **黑暗威胁**：夜晚必须有光源，否则将面临致命危险
- **精神压力**：低精神值带来影怪、幻觉等负面效果
- **资源匮乏**：有限的资源分布，鼓励探索与规划
- **怪物入侵**：定期的猎犬袭击和季节性 BOSS
- **死亡惩罚**：严厉的死亡机制，增加生存压力
- **未知恐惧**：许多机制不会明确告知，需要玩家自行探索

### 与我的世界的融合

- **资源协调**：燧石、黄金等共有资源直接使用原版内容
- **功能互补**：模组物品与原版内容的深度互动
- **风格统一**：贴图与模型符合我的世界的美学风格
- **兼容性**：考虑整合包玩家和多模组环境的需求

## 📝 设计哲学

### 为什么不是简单的内容搬运？

许多模组曾尝试将饥荒内容带入我的世界，但往往存在以下问题：

- **缺乏整体性**：只还原某个系统（如烹饪锅或精神值），缺少完整的游戏循环
- **风格不协调**：贴图像素数过高或模型质量过时
- **互动性不足**：与原版我的世界缺乏深度互动
- **目标不明确**：缺少一个清晰的游戏目标

永恒领域模组的目标是创造一个**完整的、协调的、有目标的**游戏体验。

### 关键设计问题

我们在开发过程中深入思考了以下问题：

1. **世界生成**：有限世界还是无限世界？
2. **地形设计**：超平坦还是多样地形？
3. **资源平衡**：如何协调饥荒和我的世界的资源获取方式？
4. **维度设计**：新维度还是基于主世界？
5. **难度曲线**：如何在保持硬核的同时避免过度惩罚？

这些问题的答案将在开发过程中逐步明确，我们会根据测试反馈不断调整。
