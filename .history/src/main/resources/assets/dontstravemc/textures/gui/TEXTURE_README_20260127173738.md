# Sanity Texture (sanity.png)

This file should be a 27x9 pixel sprite sheet with 3 brain icons (each 9x9 pixels):

- Position 0-9: Full brain (pink/magenta)
- Position 9-18: Half brain (partially filled)
- Position 18-27: Empty brain (outline only)

You need to create this texture file using an image editor or pixel art tool.

Recommended colors:

- Full: #FF55FF (bright magenta/pink)
- Half: #AA33AA (medium magenta)
- Empty: #555555 (dark gray outline)

Place the final PNG file at: `src/main/resources/assets/dontstravemc/textures/gui/sanity.png`
